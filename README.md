# Car Sales Data Analysis

## Overview

This project performs Exploratory Data Analysis (EDA) on a large-scale used car sales dataset containing **558,837 vehicle records** and **16 features**.

The objective is to explore vehicle market trends, pricing patterns, transmission preferences, seller performance, mileage impacts, and market valuation differences using Python and Pandas.

This project was developed as part of my Data Engineering learning journey to gain hands-on experience working with large real-world datasets.

## Key Insights

* A small number of manufacturers account for a significant share of vehicle sales.
* Automatic transmission vehicles dominate the used car market.
* Vehicle mileage has a noticeable impact on resale value.
* Selling prices vary significantly across brands and states.
* Market valuation (MMR) and actual selling prices often differ, providing useful insights into market behavior.
